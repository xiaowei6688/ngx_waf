# 兼容性说明

# 平台兼容性

本模块不向 Windows 平台提供兼容性支持。

# nginx 兼容性

本模块目前支持 nginx-1.18.0 或更新的版本。

# 模块兼容性

# ngx_http_rewrite_module

本模块与 ngx_http_rewrite_module (opens new window) 存在兼容性问题。当使用了 return 或 rewrite 指令时可能会导致本模块的检测流程被跳过。

解决方式见 waf_mode。
